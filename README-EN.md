# Turn Into Eggs
[简中](README.md)
## Data Pack Introduction
Turn Into Eggs (TIE) is a Minecraft Java Edition data pack produced by Miss & Far.<br>
This data pack adds crafting recipes for all "spawn eggs" in Minecraft.
